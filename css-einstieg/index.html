1. Wozu ist CSS da?

Webseiten bestehen aus zwei Teilen: HTML beschreibt die Struktur und den Inhalt (Überschriften, Absätze, Bilder …), und CSS beschreibt das Aussehen (Farben, Schriftgrößen, Abstände …).

Ohne CSS sieht jede Webseite gleich aus - schwarzer Text auf weißem Hintergrund. Mit CSS kannst du alles nach deinen Wünschen gestalten!

3. Selektoren - Elemente auswählen

Ein Selektor bestimmt, welche HTML-Elemente von einer CSS-Regel betroffen sind.

Element-Selektor

Der Element-Selektor wählt alle Elemente eines bestimmten Typs aus. Zum Beispiel wählt p alle Absätze (<p>) aus.

Klassen-Selektor

Mit dem Klassen-Selektor wählst du nur Elemente aus, die eine bestimmte class haben. Im HTML schreibst du class="wichtig", im CSS wählst du mit .wichtig (mit Punkt!) aus.

4. CSS wirkt auf viele Elemente gleichzeitig

Eine einzelne CSS-Regel kann mehrere Elemente auf einmal verändern. Wenn du zum Beispiel p { color: green; } schreibst, werden alle <p>-Elemente auf der Seite grün - nicht nur eins!

6. Trennung von Inhalt und Design

Ein Kernprinzip der Webentwicklung: Inhalt (HTML) und Design (CSS) sind technisch getrennt.

Stell dir einen Brief vor: Der Inhalt (was du schreibst) ist unabhängig vom Briefpapier (Farbe, Schriftart, Layout). Du kannst denselben Text auf verschiedenem Papier drucken - der Inhalt bleibt gleich, nur das Aussehen ändert sich.

Genauso funktioniert es bei Webseiten: Du änderst nicht das HTML, um das Aussehen zu ändern. Stattdessen schreibst du CSS-Regeln in einer eigenen CSS-Datei, die per <link> eingebunden wird.
